### How to use ?
- Download the main shell file 
```sh
git clone https://github.com/Gun8hoot/Deb_Discord_updater.git
```
- Add execution permission 
```sh
chmod u+x ./discord_auto-update.sh
```
- Execute the program
```sh
./discord_auto-update.sh
```
